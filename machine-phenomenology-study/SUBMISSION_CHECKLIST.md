# Submission Checklist

- [ ] Confirm manuscript title, author list, and author order
- [ ] Add affiliations and corresponding author email
- [ ] Select target venue and word-limit target
- [ ] Match target venue section structure and abstract format
- [ ] Verify exploratory vs confirmatory labeling is explicit in all sections
- [ ] Include confirmatory status statement (completed / pending)
- [ ] Include inter-rater agreement once confirmatory scoring is complete
- [ ] Attach protocol artifacts or public repository link
- [ ] Add references in target citation style
- [ ] Run final language/grammar pass
- [ ] Complete internal consistency check (metrics, thresholds, run counts)
- [ ] Export camera-ready DOCX/PDF
- [ ] Prepare cover letter tailored to target venue
- [ ] Prepare supplementary package index (protocol + raw scoring files)
